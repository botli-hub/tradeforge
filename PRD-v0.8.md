## 三、新增功能（v0.8）

### 3.1 Formula 模式

#### 3.1.1 什么是 Formula 模式

类似 TradingView 的 Pine Script DSL，适合有交易经验但不懂编程的用户。

#### 3.1.2 DSL 语法

```javascript
// 示例：双均线金叉策略
strategy("MA Cross", capital=100000, fee=0.0003)

// 参数（可在UI调节）
fast = param("快线周期", 5, 2, 50)
slow = param("慢线周期", 20, 5, 200)

// 指标计算
ma_fast = MA(close, fast)
ma_slow = MA(close, slow)
vol_ratio = volume / MA(volume, 20)

// 信号定义
entry_long = cross_above(ma_fast, ma_slow) and vol_ratio > 1.5
exit_long = cross_below(ma_fast, ma_slow)

// 止损止盈
stop_loss = 2 * ATR(14)
take_profit = 3 * ATR(14)

// 仓位
size = fixed_ratio(0.1) // 10%

// 执行
if entry_long:
  buy(size, stop=stop_loss, target=take_profit)

if exit_long:
  sell_all()
```

#### 3.1.3 内置函数库

| 分类 | 函数 |
|------|------|
| 均线 | MA, EMA, WMA, HMA |
| 动量 | RSI, MACD, KDJ, CCI, MOM |
| 波动 | ATR, BBands, STDEV |
| 成交量 | OBV, VWAP, MFI |
| 工具 | highest, lowest, cross_above, cross_below, barssince |
| 仓位 | fixed_amount, fixed_ratio, kelly, atr_sizing |
| 风控 | trailing_stop, time_stop, drawdown_stop |

### 3.2 多标的回测

#### 3.2.1 功能说明

v0.6 只支持单标的，v0.8 支持同时回测多个股票。

#### 3.2.2 资金分配模式

| 模式 | 说明 |
|------|------|
| **共享资金池** | 默认，所有标的共享初始资金，单标的最大仓位50% |
| **独立模式** | 每个标的独立计算仓位（可选） |

#### 3.2.3 多标的 UI

```
┌─────────────────────────────────────────────────────────┐
│  选择标的                                    [+ 添加]   │

│  └─────────┘ └─────────┘ └─────────┘ └─────────┘   │
│                                                         │
│  资金模式: (●) 共享资金池  ( ) 独立                  │
```
