### 6.6 页面交互说明

**页面跳转**：
```
行情页 ──[新建策略]──▶ 策略编辑器
策略页 ──[回测]─────▶ 回测页
```

**错误处理**：
- 网络超时 → Toast：「请求超时，请重试」
- 数据格式错误 → Toast：「数据解析失败」
- 策略校验失败 → 编辑器内红色提示

## 七、回测规则（完整版）

### 7.1 撮合时序

| 规则 | 说明 |
|------|------|
| 信号触发 | 收盘后检测 |
| 成交时序 | 下一根 K 线开盘价成交 |
| 买入成交价 | 开盘价 × (1 + 滑点%) |
| 卖出成交价 | 开盘价 × (1 - 滑点%) |
| 止损/止盈 | 触发后下一根 K 线开盘价成交 |

### 7.2 风控规则

| 规则 | 说明 |
|------|------|
| 止损 | 固定比例止损 |
| 止盈 | 固定比例止盈 |
| 最大持仓 | 单标的最少资金 50% |

### 7.3 限制说明（v0.6）

> **重要**：当前回测仅用于策略原型验证，不作为实盘收益承诺。
> - ❌ 不支持部分成交
> - ❌ 不处理停牌/缺失 K 线
> - ❌ 不处理复权
> - ❌ 不处理公司行为（拆股/分红）
> - ❌ 不支持做空
> - ❌ 不支持期权

### 7.4 结果一致性标准

| 维度 | 标准 |
|------|------|
| 语义一致 | 同一 Visual 策略转译为唯一 IR |
| 执行一致 | 同一 IR、同一数据、同一参数，结果必须一致 |
| 误差范围 | 指标值误差 < 0.01%，PnL 误差 < $0.01 |

## 八、数据存储设计

### SQLite 表结构

```sql
-- 策略定义
CREATE TABLE strategies (
    id TEXT PRIMARY KEY,
    name TEXT NOT NULL,
    mode TEXT NOT NULL,  -- visual
    config TEXT NOT NULL, -- JSON (IR)
    status TEXT DEFAULT 'draft',
    version INTEGER DEFAULT 1,
    created_at TEXT,
    updated_at TEXT
);

-- 策略版本历史
CREATE TABLE strategy_versions (
    id TEXT PRIMARY KEY,
    strategy_id TEXT REFERENCES strategies(id),
    version INTEGER NOT NULL,
    config TEXT NOT NULL,
    created_at TEXT
);

-- 回测记录
CREATE TABLE backtest_runs (
    id TEXT PRIMARY KEY,
    strategy_id TEXT REFERENCES strategies(id),
    symbol TEXT NOT NULL,
    timeframe TEXT NOT NULL,
    start_date TEXT,
    end_date TEXT,
    metrics TEXT,  -- JSON
    status TEXT,  -- running/completed/failed
    created_at TEXT
);

-- 成交记录
CREATE TABLE trades (
    id TEXT PRIMARY KEY,
    backtest_id TEXT REFERENCES backtest_runs(id),
    symbol TEXT NOT NULL,
    direction TEXT NOT NULL,
    entry_time TEXT,
    entry_price REAL,
    exit_time TEXT,
    exit_price REAL,
    quantity REAL,
    pnl REAL
);
```

**说明**：
- strategy_versions：保留历史版本，支持回滚
- backtest_runs：每次回测独立记录
- trades：关联到具体回测

## 十、回测指标（明确版）

### 核心指标（8项）

| # | 指标 | 说明 |
|---|------|------|
| 1 | 总收益率 | (期末资金 - 初始资金) / 初始资金 |
| 2 | 年化收益率 | 年化后的收益率 |
| 3 | 最大回撤 | 资金曲线的最大跌幅 |
| 4 | 夏普比率 | (年化收益 - 无风险利率) / 年化波动率 |
| 5 | 胜率 | 盈利交易数 / 总交易数 |
| 6 | 盈亏比 | 平均盈利 / 平均亏损 |
| 7 | 交易次数 | 总买入 + 卖出次数 |
| 8 | 平均持仓周期 | 总持仓天数 / 交易次数 |

### 展示图表

| 图表 | 说明 |
|------|------|
| 资金曲线 | 账户价值随时间变化 |
| 回撤曲线 | 从峰值的回撤幅度 |
| K线图 | 叠加买卖点标记 |
