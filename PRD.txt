TradeForge PRD v0.6
========================================

一、产品定位
----------------------------------------
v0.6 = 策略研究与模拟交易工具
目标用户: 入门用户(Visual模式)
数据源: Mock
交付: 本地桌面应用(Tauri)


二、MVP范围
----------------------------------------
- Mock数据
- 单标的
- Visual策略
- 回测 + 报告


三、版本规划
----------------------------------------
v0.6: 入门 - Visual + Mock
v0.8: 进阶 - Formula
v1.0: 专业 - Script + 富途


四、用户故事
----------------------------------------
入门用户: 拖拽条件块,5分钟看到回测


五、主流程
----------------------------------------
1.创建: 行情->新建策略->编辑器->保存
2.回测: 策略列表->选择->执行->报告
3.修改: 结果不佳->编辑->重跑


六、页面详细设计
----------------------------------------
6.1 页面结构: 5个页面
  /market /strategy /backtest /settings

6.2 行情页 /market
  - 搜索框+搜索结果
  - K线图表(lightweight-charts)
  - 周期切换:1m/5m/15m/1h/1D
  - 指标叠加:MA/EMA/BBands
  - 副图:MACD/RSI/Volume
  - 快捷:新建策略

6.3 策略页 /strategy
  - 策略卡片列表
  - 卡片内容:名称/标的/周期/状态/回测收益
  - 策略编辑器(弹窗):
    * 策略名称
    * 标的(支持多选)
    * 周期
    * 入场条件(AND组合)
    * 出场条件(OR组合)
    * 风控:止损/止盈
    * 资金:初始资金/手续费/滑点

6.4 回测页 /backtest
  - 选择策略+标的+时间区间
  - 参数设置
  - 执行按钮+进度条
  - 8项指标卡片
  - 资金曲线图+买卖点
  - 交易明细表
  - 状态:空/执行中/完成/失败

6.5 设置页 /settings
  - 回测参数:初始资金/手续费/滑点
  - 数据源:Mock模式
  - 显示:主题/语言

6.6 页面交互
  跳转: 行情->策略编辑器, 策略->回测, 回测->编辑
  共享: 当前标的/策略配置/回测结果
  错误: Toast提示


七、IR结构
----------------------------------------
{version, strategy_id, symbols, timeframe,
 indicators:[],
 conditions:{entry:{type, rules:[]}, exit:{}},
 position_sizing:{type, value},
 risk_rules:{},
 data_requirements:{},
 runtime_state:{}}


八、回测规则
----------------------------------------
- 信号:收盘后触发
- 成交:下一根K线开盘价
- 限制:不处理停牌/复权/部分成交/做空/期权
- 一致性:同一IR/数据/参数,结果必须一致


九、数据存储
----------------------------------------
表: strategies / strategy_versions / backtest_runs / trades


十、状态管理
----------------------------------------
Store: app / strategy / backtest / market
任务状态: idle/running/completed/failed


十一、指标(8项)
----------------------------------------
总收益/年化/最大回撤/夏普/胜率/盈亏比/次数/平均持仓


十二、性能目标
----------------------------------------
MacBook Air,单标,5年日线,2均线:
- 回测<5秒
- 交互<100ms


十三、里程碑验收
----------------------------------------
M1: 项目+MOCK K线
M2: Visual编辑器
M3: IR转译层
M4: 回测引擎
M5: 集成优化


十四、技术选型
----------------------------------------
React+TypeScript / Zustand / lightweight-charts
react-flow / Tauri 2.0 / Python FastAPI


十五、后续版本
----------------------------------------
v0.8: Formula/多标的
v1.0: Script/富途/国际化
